About Page

The User Task Management System is a web-based application developed using Servlets that provides a simple and secure platform for managing users and their tasks. It consists of two modules: Admin and User, each with specific roles and functionalities.

In the Admin module, the administrator can log in securely and manage users by approving newly registered accounts or blocking inactive or suspicious ones. The admin also has the ability to view tasks created by different users, ensuring better monitoring and control. This module ensures that only authorized and verified users are allowed to access the system.

In the User module, individuals can register themselves on the portal. However, their access becomes active only after approval by the admin. Once logged in, users can add new tasks, update existing tasks, delete completed tasks, and retrieve their to-do list at any time. This provides a personal task management environment that is easy to use and effective in organizing daily activities.

The system is designed with simplicity and clarity, using HTML forms for input and Servlets for business logic. Overall, this project demonstrates concepts of user authentication, session handling, and CRUD operations in a real-world application scenario.
